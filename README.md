# 🚌 CampusRide

Smart transport management system for university students.

## Features
- Student registration & dashboard
- Cashless card system with scan & ride
- Admin panel (users, routes, payments, analytics)
- Driver panel (route, students, delay reporting)
- Routes for Painesville, Broadville & Baton Rougeville

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to vercel.com → Import project → Select this repo → Deploy

## Run Locally
```bash
npm install
npm start
```
